Performance Attribution in Global Performance Evaluation

Created for AlchemistsAcademy by MacLane Wilkison

Multicurrency Returns

  • Local currency return: r = p + d
  • Base currency return: r = p + d + s(1 + p + d)
    • Total return in base currency = Capital gain component + Yield component + Currency component
'p' is the capital gain return. 'd' is the dividend yield. 's' is the percentage exchange rate movement on the estimated return in the base currency.

Performance Attribution

  • Security selection: r = I + (p - I) + d + c
    • Total portfolio return = Market return component + Security selection contribution + Yield component + Currency component
  • Asset allocation
    • Total portfolio return = International benchmark return + Market allocation contribution + Currency allocation contribution + Yield component + Security selection contribution
'I' is the local currency return on the corresponding market index. Asset allocation: a manager's relative performance can be attributed to the following factors: (1) a market allocation different from that of the index (market allocation contribution); (2) A currency allocation different from that of the index (currency allocation contribution); (3) Superior security selection (Security selection contribution)
